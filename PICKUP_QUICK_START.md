# 🎯 Самовывоз - Краткая инструкция

## ✅ Что работает

### Клиент при оформлении заказа видит:

```
📦 Способ получения
┌──────────────────┬──────────────────┐
│ 🚚 Доставка      │ 🏪 Самовывоз     │
└──────────────────┴──────────────────┘
```

**При выборе Самовывоза:**
- 📍 Адрес ресторана: ул. Абдымомунова, 265, Бишкек 720040, Кыргызстан
- 📞 Телефон: +996 (312) 62-55-55
- Поле адреса доставки скрывается (не требуется)

**При выборе Доставки:**
- Поле адреса показывается (требуется заполнить)

## 🔄 Как данные передаются

### Фронтенд → Backend
```json
{
  "meta_data": [
    {
      "key": "order_type",
      "value": "pickup" // или "delivery"
    }
  ]
}
```

### Backend → WordPress Admin
Плагин отображает:
```
Тип заказа: Самовывоз ✅
Или
Тип заказа: Доставка ✅
```
