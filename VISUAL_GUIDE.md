         │   value: "pickup"
         │ }
         │
         ▼
┌───────────────────────┐
│ WordPress REST API    │
│ /wp-json/wc/v3/      │
│ /orders               │
└────────┬──────────────┘
         │
         │ Плагин обрабатывает
         │ meta_data
         │
         ▼
┌──────────────────┐

│ meta_key:        │

         │
         │ Админ-панель

         │
         ▼

│  Админ-панель    │

│                  │

└──────────────────┘

## ⚡ Быстрая настройка

### Шаг 1: Обновите адреса (Checkout.tsx)
```typescript
// НАЙТИ:
const RESTAURANT_ADDRESS = 'ул. Абдымомунова, 265, Бишкек 720040, Кыргызстан';
const RESTAURANT_PHONE = '+996 (312) 62-55-55';

// ЗАМЕНИТЬ на:
const RESTAURANT_ADDRESS = 'Ваш реальный адрес';
const RESTAURANT_PHONE = 'Ваш реальный телефон';
```

### Шаг 2: Активируйте плагин
```
WordPress → Плагины → Order Type Display → Активировать
```

### Шаг 3: Тестируйте!
- Создайте заказ с типом "Доставка"
- Создайте заказ с типом "Самовывоз"
- Проверьте информацию в админ-панели

---

## 🎯 Результаты

После интеграции:

✅ **Клиент** видит два варианта получения заказа  
✅ **Клиент** может выбрать способ доставки  
✅ **Форма** адаптируется в зависимости от выбора  
✅ **Заказ** содержит информацию о типе доставки  
✅ **Админ** видит тип заказа с иконкой 📦 или 🚚  
✅ **Админ** может редактировать тип прямо в заказе  
✅ **Письма** содержат информацию о типе доставки
