## 🆘 Если что-то не работает

### Ошибка: `meta_data` не сохраняется

- [ ] Проверить версию WooCommerce (совместимость)
- [ ] Проверить права доступа в БД
- [ ] Переактивировать плагин

### Ошибка: Тип не видна в админ-панели

- [ ] Убедиться, что плагин активирован
- [ ] Очистить кэш WordPress (если есть)
- [ ] Обновить страницу заказа

### Ошибка: Адрес рестора не показывается

- [ ] Проверить константу `RESTAURANT_ADDRESS` в `Checkout.tsx`
- [ ] Проверить условие `orderType === 'pickup'`
- [ ] Проверить консоль браузера на ошибки

### Ошибка: Форма не отправляется

- [ ] Проверить валидацию адреса для доставки
- [ ] Проверить номер телефона
- [ ] Проверить консоль на ошибки API

## 📞 Контрольные вопросы

1. **Q:** Где администратор видит тип заказа?
   - **A:** В админ-панели WooCommerce, в деталях заказа и в Dashboard (список заказов)

2. **Q:** Что происходит если клиент выбирает Самовывоз?
   - **A:** Поле адреса скрывается, показывается адрес ресторана

3. **Q:** Как плагин WordPress узнает о типе?
   - **A:** Из поля `meta_data` с ключом `order_type`

4. **Q:** Какой адрес сохраняется в заказе при самовывозе?
   - **A:** Адрес ресторана (ул. Абдымомунова, 265...)

5. **Q:** Что делать если нужно изменить адрес ресторана?
   - **A:** Изменить константу в `src/components/Checkout.tsx`
