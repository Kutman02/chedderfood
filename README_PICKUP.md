## 🔌 ПЛАГИН WORDPRESS

**Файл:** `wordpress-plugins/order-type-display.php`

Плагин отображает тип заказа в админ-панели WooCommerce:

```php
<?php
add_action('woocommerce_admin_order_data_after_order_details', function($order){
    $order_type = '';
    foreach ($order->get_meta_data() as $meta) {
        if ($meta->key === 'order_type') {
            $order_type = $meta->value;
            break;
        }
    }

    if ($order_type) {
        $label = $order_type === 'pickup' ? 'Самовывоз' : 'Доставка';
        echo '<p><strong>Тип заказа:</strong> ' . esc_html($label) . '</p>';
    }
});
?>
```

**Плагин остается без изменений - используется как предоставлен.**

## 🔧 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Тип заказа не видна в админ-панели
- Убедитесь что плагин активирован: WordPress → Плагины → Order Type Display → Активировать

### Адрес ресторана не показывается
- Откройте консоль (F12) и ищите ошибки
- Проверьте что `orderType === 'pickup'` работает

### Форма не отправляется при доставке
- Проверьте что поле адреса заполнено
- Проверьте что телефон верный формат

### Как изменить адрес ресторана
- Откройте `src/components/Checkout.tsx`
- Найдите константы в начале файла
- Измените `RESTAURANT_ADDRESS` и `RESTAURANT_PHONE`
