## 🚀 КАК ИСПОЛЬЗОВАТЬ

### Для клиента
1. Откройте корзину → Оформить заказ
2. Выберите "Доставка" или "Самовывоз"
3. При самовывозе: видите адрес ресторана
4. При доставке: вводите свой адрес
5. Создайте заказ

### Для администратора
1. Откройте Dashboard или WooCommerce
2. Видите тип заказа (синий/зеленый баннер)
3. Кликните на заказ для деталей
4. Видите полную информацию о типе
5. Обслуживайте заказ согласно типу
